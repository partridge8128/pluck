Views

TreeHierarchical tree (default). Best for exploring shape.
TableTabular layout — auto-enabled when the current result is an array of objects with consistent shape. Click a column header to sort. Use project() in Filter mode to pick specific fields with friendly names. Copy CSV / Save CSV in the toolbar.
RawPretty-printed JSON text — useful for quick visual review or copy.
SwitchingTabs in the result pane header, or command palette (Ctrl+K) → "Tree / Table / Raw view". Active view is persisted across reloads.

Parser modes

StrictDefault. Standard JSON. Errors are real errors.
JSONCJSON + // + /* */ comments + trailing commas. Click JSONC in the source pane after a strict parse fails, or open a .jsonc file.
JSON5JSONC + unquoted keys + single-quoted strings + hex numbers + Infinity/NaN + leading +/. on numbers + line continuations. Click JSON5 in the source pane, or open a .json5 file.
SwitchingActive parser shown as a badge in the topbar — click the × on it to return to strict. Or use the command palette (Ctrl+K) → "Strict / JSONC / JSON5".
Output is strictFormat, Minify, Copy, and Save always emit standard JSON — comments and JSON5 syntax are lost on round-trip. The textarea text is preserved otherwise.

Privacy & data

Local onlyEverything runs in your browser. The page makes zero outbound requests after it loads — no analytics, no fonts CDN, nothing.
Session memory
Off by default. When on, your pasted JSON / mode / query / view state are stored in localStorage (this browser only). Toggle off to clear instantly.
Saved commandsThe ★ Saved feature persists independently — those are explicit saves you made, not auto-captured. Use the buttons below to wipe everything.
Clear data Removes session state, saved commands, and the persistence opt-in flag.

Filter mode (JavaScript)

⚠ Heads-upFilter expressions are evaluated as JavaScript in your browser via new Function(). Do not paste filter expressions from sources you don't trust — they can run arbitrary code (read your DOM, call APIs, etc.). Same trust level as the browser DevTools console.
